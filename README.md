# Analisis Turnover Karyawan Menggunakan Exploratory Data Analysis (EDA)

Proyek ini melakukan analisis data karyawan (HR Analytics) untuk memahami mengapa karyawan keluar dari perusahaan (employee turnover). Dataset yang digunakan berisi berbagai atribut terkait kondisi kerja karyawan seperti tingkat kepuasan, gaji, departemen, dan status apakah karyawan keluar atau tidak.

Tujuan utama analisis ini adalah mengidentifikasi faktor yang memengaruhi keputusan karyawan untuk meninggalkan perusahaan, sehingga perusahaan dapat membuat strategi untuk meningkatkan retensi karyawan.
